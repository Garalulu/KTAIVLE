- numpy, pandas, SQL, ...
- EDA
    - 개별변수 

    |  | 그래프 | 통계량 |
    |---|---|---|
    |숫자|히스토그램, kdeplot, 박스플롯|min, max, mean, std, 사분위수|

    - 가설 x -> y
        - 숫자 -> 숫자 // 상관분석, 상관계수

            - p-value < 0.05: 상관계수가 유의미함
        - 범주 -> 숫자
            - 그래프: barplot
            - t통계량: 두 평균의 차이
            - f통계량: 전체평균 vs 개별평균 차이
