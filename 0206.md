#### 요약
- 분석할 수 있는 정보의 종류 2가지 // 숫자, 범주
- 두 가지 종류의 정보가 특별한 구조를 가져야 함
- 기본 구조: 2차원
- 행: 분석단위, 데이터 한 건 한 건
- 열: 정보, 변수, 요인(x, feature), 결과(y, target, label)
- 데이터 구조를 다루는 패키지: Numpy, pandas
