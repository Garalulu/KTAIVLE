### 1교시
## Grid & Random Search
- 내부적으로 K-Fold Cross Validation을 위해 cv값을 지정하므로 실제 수행되는 횟수는 파라미터 조합 수 * cv값
```python
model = RandomizedSearchCV(model_dt,      # 기본 모델
                           param,         # 파라미터 범위
                           cv=5,          # k-fold CV 폴더 개수
                           n_iter=20,     # 랜덤 시도 횟수
                           scoring='r2')  # 사용할 평가지표
```
